# Development of Standard Cell Library using 45nm CMOS Technology

> Bachelor of Engineering Major Project | Electronics and Communication Engineering

## Overview

This repository presents the complete design and development of a **Standard Cell Library (SCL)** using **45nm CMOS Technology** for ASIC Physical Design.

The project focuses on developing a high-performance standard cell library by designing and characterizing various combinational and sequential standard cells. The complete design flow includes architecture planning, beta ratio optimization, schematic design, layout development, physical verification (DRC/LVS), parasitic extraction, characterization, and performance analysis under different Process, Voltage and Temperature (PVT) conditions.

This project was developed using the **Cadence Virtuoso Design Suite** with GPDK45 technology.

---

## Project Objectives

- Design a Standard Cell Library using 45nm CMOS technology
- Optimize beta ratio for improved delay performance
- Develop layouts for standard cells
- Perform DRC and LVS verification
- Generate extracted views
- Perform delay characterization
- Analyze performance under different PVT corners
- Evaluate Setup Time and Hold Time for sequential cells
